Тут ви знайдете демонстрацію двох моїх проектів: генерація випадкового пароля та генерація випадкового переможця . Ці проекти демонструють мої навички роботи з взаємодією з сервером і обробкою асинхронних запитів

Генератор паролю

«Генератор паролю» – javascript проект, розроблений для створення надійних та унікальних паролів.
Проект взаємодіє із сервером random.org для генерації випадкового пароля.
Цей проект дозволив ознайомитися з принципами роботи з клієнт-серверною архітектурою, надсиланням HTTP-запитів та обробкою відповідей від сервера.

Випадковий переможець

Ще один javascript проект, який дозволяє проводити випадкову лотерею серед учасників.
Так само працює із сервером random.org для отримання випадкового учасника-переможця.
Загалом проект демонструє навички роботи з AJAX-запитами.
